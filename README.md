# Часы и будильник

Многофункциональное приложение с часами, будильником, таймером, секундомером и мировым временем.

## Что умеет

- **Часы** - цифровые, обновляются каждую секунду с датой и днём недели
- **Будильник** - можно добавить несколько, включать и отключать отдельно
- **Таймер** - с вводом часов/минут/секунд и ползунком быстрой установки
- **Секундомер** - с засечкой кругов
- **Мировое время** - текущее время в Москве, Лондоне, Нью-Йорке и Токио

## Технологии

HTML, CSS, JavaScript (без библиотек), localStorage, Web Audio API

## Запуск

Открыть `index.html` в любом браузере. Для звука будильника нужен файл `sounds/alarm.mp3`.
